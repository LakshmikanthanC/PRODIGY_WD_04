## 📌Installation

Follow the below steps to run the application,

**Step 1:** Run this command in your terminal to make a clone

```
gh repo clone PortFolio/lak
```
or 
Download the ZIP file

**Step 2:** Change the path to
```
cd lak
```
**Step 3:** Vite Installation
```
npm install vite
```
**Step 4:** 
```
npm run dev
```
